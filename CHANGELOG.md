# Changelog

## v1.1.0 (2026-06-19)

### ✨ 新功能
- **本地 T2I 渲染**：支持 `t2i_mode` 配置项，可选 `network`（远程服务）或 `local`（本地 Docker）
- **图片卡片重构**：全新竖向布局，总投喂独占一行，礼物+弹幕一行
- **盲盒统计卡片化**：概览区居中显示盈亏+电池，明细区 2 列网格
- **大航海头像框**：舰长用户头像自动添加 B站 CDN 头像框
- **礼物单位显示**：所有价值数字后加「电池」单位

### 🐛 修复
- 盲盒概览盈亏数字正负颜色正确显示（绿盈红亏）
- 盲盒明细卡片添加淡紫底纹，避免纯白
- `html_render` 添加 15s 超时，防止 T2I 服务卡死
- GitHub Actions 自动构建 t2i-local 镜像到 GHCR

### 🔧 优化
- Docker 镜像体积从 1.5GB → ~512MB（改用 python:3.12-slim + 仅 Chromium）
- 礼物底纹加深+阴影效果
- 盲盒大框去掉红绿底色，保持中性
